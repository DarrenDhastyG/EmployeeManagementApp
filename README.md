Spring Boot <br />
=============================
mvn clean <br />
mvn package <br />
java -jar employeeManagement-0.0.1-SNAPSHOT.jar <br />

Angular <br />
==============================
npm install <br />
npm install -g @angular/cli <br />
ng serve <br />

To access the app, http://localhost:4200 once both backend and frontend are running.
